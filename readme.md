# BuyAuto-Test-Infomaximum

BuyAuto — это веб-приложение, разработанное в рамках тестового задания компании Infomaximum. Приложение позволяет просматривать каталог автомобилей, добавлять понравившиеся модели в избранное, сортировать по различным параметрам и управлять выбором комплектации.

---

## Основной функционал

**Каталог автомобилей**  
   - Отображение списка машин с изображением, описанием, ценой и годом выпуска.  
   - Возможность сортировки по названию, цене и году выпуска.  
   - Поиск по названию автомобилей.  

**Избранное**  
   - Добавление/удаление автомобилей в избранное.  
   - Отдельная страница с избранными автомобилями.  
   - Динамический счётчик количества позиций с корректным склонением.  

**Карточка автомобиля**  
   - Картинка, имя, краткое описание, год и цвет.  
   - Цена, кнопка выбора комплектации и удаление из избранного.  
   - Адаптивный вывод текста с `ellipsis`, сохранением стиля при переполнении.  

**Интерактивные элементы**  
   - Кнопки с изменением цвета при наведении и нажатии.  
   - Иконка удаления сменяется на Hover и Pressed варианты.

## Стек технологий

- **Язык**: TypeScript
- **Фреймворк**: React
- **Стилизация**: CSS Modules
- **Управление состоянием**: MobX
- **Сборка**: Vite
- **API генерации данных**: Mock API (`generated.ts`)
